# Java 基础练习项目集

## 项目简介
这是一个 Java 基础练习项目集合，包含多个小型实践项目，用于巩固 Java 编程基础和高级特性。

## 项目列表

### 🧩 华容道游戏
- 经典华容道益智游戏的 Java 实现
- 包含图形界面和游戏逻辑
- 支持关卡选择和步数统计

### 🃏 斗地主发牌器
- 模拟斗地主游戏的发牌系统
- 实现洗牌、发牌、排序功能
- 支持玩家手牌展示

### 💬 聊天室（有 Bug）
- 基于 TCP 协议的多人聊天室
- 实时消息发送和接收
- **目前存在一些问题，希望有大佬能指导修复**

## 技术栈

### 基础技术
- Java SE
- Swing GUI（华容道）
- TCP 网络编程（聊天室）

### Java 高级特性练习
- **反射（Reflection）**
- **注解（Annotation）**
- **动态代理（Dynamic Proxy）**
